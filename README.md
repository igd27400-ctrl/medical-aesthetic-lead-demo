# 医美咨询线索分层 Demo

本项目用于展示一个小闭环：

客户咨询内容 → 信息提取 → 意向分层 → 销售跟进建议 → 表格沉淀

## 使用方式

直接双击 `index.html`，浏览器打开即可运行。

## 当前功能

- 提取咨询项目、城市、预算、时间、顾虑
- 判断高/中/低意向
- 生成销售跟进建议
- 加入线索表
- 导出 CSV

## 合规说明

本 Demo 只做咨询信息整理和销售辅助，不做医疗诊断、不承诺疗效、不替医生提供医疗建议。
